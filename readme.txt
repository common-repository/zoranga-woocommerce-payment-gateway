== Description ==

This is a Zoranga Payment Gateway for WooCommerce.

Zoranga primarily converts your airtime to cash, with this plugin, you can securely sell your products and subscriptions online while receiving airtime as payment.

Integrate this plugin into your woocommerce website is very simple and our support system is always there to guide you via call, SMS or Whatsapp 0806 723 8359.

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

If on the off-chance you do encounter issues with the shop/category pages after an update you simply need to flush the permalinks by going to WordPress > Settings > Permalinks and hitting 'save'. That should return things to normal.
